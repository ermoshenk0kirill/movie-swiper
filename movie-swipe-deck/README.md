# Movie Swipe Deck

React-приложение с интерфейсом свайпа карточек фильмов.
Карточки можно свайпать влево или вправо для взаимодействия с фильмами.

## Технологии

* React
* Vite
* TypeScript
* Tailwind CSS
* Material UI
* react-tinder-card
* Zustand

## Установка

```bash
npm install
```

## Запуск проекта

```bash
npm run dev
```

Приложение будет доступно по адресу:

```
http://localhost:5173
```

## Сборка проекта

```bash
npm run build
```
